# iZsh AI 自动确认 - 快速使用指南

## ⚠️ 重要：正确的使用方式

### ❌ 错误方式（AI 不会生效）
```bash
# 在普通终端或直接运行
claude-code
```

### ✅ 正确方式（AI 自动确认生效）

#### 步骤 1：启动 iZsh
```bash
# 方法 1：从应用程序启动
open ~/Applications/iZsh.app

# 方法 2：从命令行启动  
~/.local/bin/izsh
```

#### 步骤 2：在 iZsh 中运行 Claude Code
```bash
# 在 iZsh 提示符下输入
[iZsh] ~% claude-ai
```

## 🔧 当前问题：AI 没有响应

**原因分析**：
1. ❌ 您可能直接运行了 `claude-code`（包装器未启动）
2. ❌ 不在 iZsh 终端中

**临时解决方案**：
1. 手动按 `1` 并回车（选择 Yes）
2. 退出 Claude Code
3. 重新按照上面的正确方式启动

## 💡 验证是否在 iZsh 中

正确的提示符应该是：
```bash
[iZsh] zhangzhen@zhangzhendeMacBook-Pro:~%
```

如果是：
```bash
zhangzhen@zhangzhendeMacBook-Pro ~ %
```
说明您在普通 zsh 中，不是 iZsh！
