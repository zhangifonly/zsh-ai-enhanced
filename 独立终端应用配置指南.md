## 方式 2️⃣: Terminal 配置文件

### 配置步骤

1. **打开 Terminal 偏好设置**
   ```
   Terminal → 偏好设置 (Preferences) → 描述文件 (Profiles)
   ```

2. **导入 iZsh 配置文件**
   - 点击左下角的 **⚙️ 齿轮图标**
   - 选择 **导入 (Import)**
   - 选择文件：`~/Library/Application Support/Terminal/iZsh.terminal`

3. **设置为默认**（可选）
   - 在配置文件列表中选择 **iZsh**
   - 点击底部的 **默认 (Default)** 按钮

4. **使用 iZsh 配置文件**
   - **新建窗口**: `⌘ + N` → 自动使用 iZsh（如果设置为默认）
   - **手动选择**: `Shell → 新建窗口 → iZsh`

### 配置文件特点
- ✅ 深色主题
- ✅ Menlo 字体 14号
- ✅ 120列 × 36行
- ✅ 自动启动 iZsh
- ✅ 自定义窗口标题
- ✅ 光标闪烁
- ✅ 可视化响铃

## 💡 高级技巧

### 创建别名

在标准 zsh 中创建别名快速启动：

编辑 `~/.zshrc`：
```bash
# iZsh 快速启动别名
alias izsh='open ~/Documents/ClaudeCode/zsh/zsh/iZsh.command'
```

然后：
```bash
source ~/.zshrc
izsh  # 快速启动
```

## ❓ 故障排查

### 问题 1: 双击没反应

**解决**：
```bash
# 确保有执行权限
chmod +x ~/Desktop/iZsh.command
```

### 问题 2: 启动后立即退出

**解决**：
- 检查 `~/.local/bin/izsh` 是否存在
- 运行 `./start_izsh.sh` 查看详细错误

### 问题 3: AI 功能不工作

**解决**：
- 检查 `~/.izshrc` 配置
- 确认 API 密钥正确
- 查看网络连接
